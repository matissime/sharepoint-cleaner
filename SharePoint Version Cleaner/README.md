# SharePoint Version Cleaner

This folder contains the main PowerShell script and user guide for cleaning up file versions in SharePoint Online document libraries.

## Files

- **SPOVC.ps1**: The primary script for removing old file versions from SharePoint Online. Features a Windows Forms GUI for easy use.

## Usage

1. Open PowerShell as Administrator.
2. Navigate to this folder.
3. Run the script:
   ```powershell
   .\SPOVC.ps1
   ```
4. Follow the on-screen instructions and refer to the guide for details.

## Notes
- Ensure you have the required permissions and modules installed (`PnP.PowerShell`).
- Logs are saved in the `Logs` subfolder.
