# Backup Versions

This folder contains legacy or alternative versions of the SharePoint Version Cleaner scripts. These are provided for reference, backup, or compatibility purposes.

## Scripts

- **SPOVC.ps1**: An earlier version of the SharePoint file version cleaner script. Use this if you encounter issues with the main version or need to reference previous logic.
- **SPOVC-GraphPicker.ps1**: A variant that utilizes the Microsoft Graph API and provides a graphical interface for selecting SharePoint sites. Useful for scenarios where Microsoft Graph is preferred or required.

## Notes
- These scripts may not include the latest features or bug fixes found in the main script.
- Use with caution and always test in a safe environment first.
